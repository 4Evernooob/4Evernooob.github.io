用github pages搭建一个hexo博客

相关的教程网上有很多了 ，我就写一下重点和一些需要注意的地方。

  1. 前期准备
    注册github账号,创建名称为【你的github用户名.github.io】的repository。
    安装node.js和git环境

  2. 安装hexo，打开git bash,输入

“Generating public/private rsa key pair.
Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):<回车就好>

Enter passphrase (empty for no passphrase):<输入加密串>
Enter same passphrase again:<再次输入加密串>

Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.

$ git config –global user.name “xxx”//用户名
$ git config –global user.email “xxx@mail.com”//填写自己的邮箱

$ hexo clean
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

theme: next
#
deploy:
type: git
repository: https://github.com/你的github用户名/你的github用户名.github.io.git
branch: master
#
language: zh-Hans

https://github.com/你的github用户名/你的github用户名.github.io```上面访问你的博客了。

  • 有关NexT主题的其他详细设置，都可以在官网找到答案。
    1. 绑定独立域名
  • 首先你要去购买一个域名，比如在万网上拍下一个域名后，进入阿里云的控制台–左侧侧菜单栏点击域名–找到你的域名在屏幕靠右侧点击”解析”。
    添加一条解析，记录类型为CNAME，主机记录为@，记录值为“你的github用户名.github.io”，保存。
  • 在 本地hexo文件夹\source 目录下新建一个txt文档，内容输入你的独立域名（不需要加http或者www），然后重命名该txt为CNAME（不要文件后缀名）。
  • 在git bash中输入hexo d -g更新项目后,就成功绑定了你的独立域名。
