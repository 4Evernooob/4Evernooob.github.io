绑定独立域名

首先你要去购买一个域名，比如在万网上拍下一个域名后，进入阿里云的控制台–左侧侧菜单栏点击域名–找到你的域名在屏幕靠右侧点击”解析”。
添加一条解析，记录类型为CNAME，主机记录为@，记录值为“你的github用户名.github.io”，保存。
在”本地hexo文件夹\source”目录下新建一个txt文档，内容输入你的独立域名（不需要加http或者www），然后重命名该txt为CNAME（不要文件后缀名）。
